# 🧮 FJ Calculator Project

A responsive and interactive **web-based calculator application** built using **HTML, CSS, and JavaScript**.  
This project demonstrates core frontend engineering principles, including DOM manipulation, event-driven programming, and clean UI design architecture.

## 🌐 Live Demo
🔗 https://muhammad-fahad-javed.github.io/calculator.github.io/


## 🎯 Project Overview
This project was developed to strengthen fundamental frontend development skills by building a fully functional calculator with a focus on:

- Logical problem-solving  
- Interactive UI behavior  
- Clean and maintainable code structure  
- Practical implementation of JavaScript fundamentals  

It serves as a foundational project in my frontend development journey.


## ⚙️ Key Features
- ➕ Basic arithmetic operations (Addition, Subtraction, Multiplication, Division)  
- ⚡ Real-time interactive calculations  
- 🧼 Clear and reset functionality for better usability  
- 🎯 Minimal, clean, and user-friendly interface  
- 📱 Fully responsive design for all screen sizes  

## 🛠️ Technology Stack

- **HTML5** – Semantic structure and layout  
- **CSS3** – Styling, responsiveness, and UI design  
- **JavaScript** – Core logic, DOM manipulation, and interactivity  


## 🧠 Key Learning Outcomes
Through this project, I gained practical experience in:

- DOM manipulation and element selection  
- Event-driven programming in JavaScript  
- Structuring interactive web applications  
- Improving UI/UX through minimal design principles  
- Writing clean and maintainable frontend code
